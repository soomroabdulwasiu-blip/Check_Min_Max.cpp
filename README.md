## Check_Min_Max.cpp
This Repository contains simple C++ Programs to Check the number is minimum and maximum

## Features

-First User Enter the number of elements:
-it must be greater than or equal to 0 otherwise, Please enter a positive integer for the number of elements. This Condition is given by if else condition.
-and user enter a number of elements:
-and I use for loop and check weather the number is maximum and maximum.
